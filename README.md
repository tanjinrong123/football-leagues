# myvue #

- 用到的技术栈
- 数据库：mysql
- 后端：[ back-end ]node + express + mysql(node数据库模块)
- 前端: [ src ]: vue+ elment-ui + iview + VueAwesomeSwiper + axios + webpack

> A Vue.js project

## 前言 ##

- 后端负责提供接口，操作mysql数据库提供的数据和状态。
- 前端负责调用接口，展示数据。

## 功能 ##

- [x] 五大联赛各联赛、俱乐部、球员的数据查询
- [x] 用户登录/注销/注册
